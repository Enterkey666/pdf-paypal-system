PDF処理 & PayPal決済リンク発行システム
===================================================

【インストール方法】
1. setup.bat をダブルクリックしてセットアップを開始してください。
2. 画面の指示に従って、デスクトップにショートカットを作成するかどうかを選択します。
3. アプリケーションを今すぐ起動するかどうかを選択します。

【手動起動方法】
1. コマンドプロンプトを開き、このフォルダに移動します。
2. 「python app.py」コマンドを実行します。
3. ブラウザで http://localhost:5000 にアクセスします。

【必要環境】
- Python 3.7以上
- 必要なライブラリ（自動インストールされます）

※ 初回起動時に必要なライブラリが自動的にインストールされます。
※ 詳細な説明はREADME.mdファイルをご覧ください。
